★★★　Asset Installation Instructions　★★★

〇 Installing the VFX Assets:
1. Open the "VFX" Folder found in the same file directory as this manual.
2. Copy (Right-click > "Copy") the entire "effects" folder.
3. Open your RPG Maker MZ project's folder.
4. Paste the "effects" folder there.

〇 Installing the SFX Assets:
1. Open the "SFX" Folder found in the same file directory as this manual.
2. Copy (Right-click > "Copy") all the files in the folder.
3. Open your RPG Maker MZ project's folder, then navigate to the "audio" > "se" Folder.
4. Paste all the sound effect files there.

〇 Installing the plugin/lib files:
1. Open the "libs" Folder found in the same file directory as this manual.
2. Copy (Right-click > "Copy") all the files in the folder.
3. Open your RPG Maker MZ project's folder, then navigate to the "js" > "libs" folder.
4. Paste all the files there.

5. Open the "plugins" Folder found in the same file directory as this manual.
2. Copy (Right-click > "Copy") all the files in the folder.
3. Open your RPG Maker MZ project's folder, then navigate to the "js" > "plugins" folder.
4. Paste all the files there.

〇 Installing the RPG Maker MZ Presets:
1. Open the RPG Maker MZ Demo project AND your RPG Maker MZ project in SEPARATE instances of RPG Maker MZ.
2. Open the Database for both projects, and navigate to the "Animations" tab.
3. Copy the animation entries from the Demo project.
4. Paste the animation entries into your project.
